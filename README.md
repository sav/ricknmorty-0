# Rick and Morty API Viewer

A simple C++/Qt application to view characters and episodes from the Rick and Morty API.

## Installation and Compilation on Ubuntu 24.04

### Dependencies

First, ensure you have the necessary Qt6 development packages and `qmake` installed:

```bash
sudo apt update
sudo apt install qt6-base-dev qt6-base-dev-tools build-essential
```

### Compilation

Navigate to the project root directory and compile using `qmake` and `make`:

```bash
qmake ricknmorty.pro
make
```

### Running the Application

After successful compilation, you can run the application:

```bash
./run.sh
```
